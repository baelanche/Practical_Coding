* Software Testing

## Software Testing
소프트웨어에서 발생할 수 있는 오류들을 탐지

### Static Testing
프로그램을 실행하지 않고 소스코드를 분석하는 기법

### Dynamic Testing
소프트웨어를 실행하여 동적으로 소스코드를 분석하는 기법

### VS
|정적테스팅|동적테스팅|
| ______ | ______ |
| 수행속도 빠름|수행속도 느림|
|수행노력 적음|수행노력 많음|
|정밀검증 힘듦|정밀검증 가능|


### White Box
소스코드를 확인하는 테스팅

### Black Box
소스코드를 확인하지 않는 테스팅

## Coverage
얼만큼의 테스팅을 했는지 나타내는 수치이다
